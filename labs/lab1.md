# Lab 1 - Host Discovery with Nmap

## Objective
Discover active hosts in the network.

## Command Used
nmap -sn 192.168.1.0/24

## Explanation
- `-sn` performs a ping scan
- It checks which hosts are online without scanning ports

## Result
The scan identifies active devices in the network.

## Notes
Host discovery helps identify targets before performing deeper scans.
